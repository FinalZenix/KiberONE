# Flappy Bird Project

## Project Overview
This is the main project for the Module 2 (Advanced Python) group.
We are using `pgzrun` (Pygame Zero) to recreate Flappy Bird.

## Assets
- **Images:** stored in standard `images/` directory relative to the script.
- **Sounds:** stored in `sounds/` directory.

## Code Versions
- `snapshot_2025-12-13.py`: State of the code after lesson on 13.12.2025. Includes:
    - Gravity (Velocity loop)
    - Basic Jump (Key Input)
    - Drawing Bg/Ground/Bird
- `final_version.py`: The target final code (Teacher Reference). Includes:
    - Pipes & Collision
    - Score
    - Game States (Menu, Playing, Dead)
    - Rotation Animation
